Aula 1

C++ é uma linguagem compilada

Processo de compilação:
1) Pré-processador (Codigo Fonte)
2) Compilador e Assembler (Codigo Assembly)
4) Linker (Codigo objeto)

Toda alteração de código faz com que ele precise ser compilado novamente

É uma linguagem Case Sensitive

As variaveis precisam ser declaradas antes de serem usadas

int - 4 bytes
char - 1 byte
string - especial
bool - 1 byte
float - 4 bytes

Variavel tem:

Nome da Variavel | Conteudo da Variavel | Endereço na Memoria

-- 

stdafx.h - header que inclue bibliotecas
iostream - inclui funcoes de entrada e saida
string - permite trabalhar com strings
double - 8 bytes

Ctrl + Shift + B = compilar
F5 = debuggar 
Ctrl + F5 = Somente executar
